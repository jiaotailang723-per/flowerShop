# flower_shop
#### 介绍
“朝花夕拾（花店商城）”是基于Java Web技术构建的电子商务平台，专注于花卉及相关产品的在线销售。平台整合了商品展示、购物车管理、订单处理、支付配送等核心功能，同时提供强大的后台管理系统，支持商品管理、订单跟踪、客户管理及类目管理等操作，助力花店实现数字化转型，提升运营效率与用户购物体验。

#### 软件架构
1. **架构模式**：采用B/S架构，分为表示层、业务逻辑层、数据层三层架构
2. **技术栈**
   - 前端：HTML、CSS、JavaScript、Font Awesome、FlexSlider、Layer.js
   - 后端：Java、Servlet、JSP（MVC模式）
   - 数据库：MySQL（支持事务管理与索引优化）
   - 开发工具：IDEA、Eclipse、MySQL Workbench
   - 测试工具：JMeter、Selenium、禅道、Appscan
3. **核心模块**
   - 前台模块：用户注册登录、商品展示、商品搜索、购物车管理、订单管理、支付与配送
   - 后台模块：商品管理、订单管理、客户管理、商品类目管理

#### 功能特点
1. **用户体验优化**
   - 现代化卡片式设计，支持响应式布局（适配移动端与桌面端）
   - 表单实时验证（用户名、手机号、密码格式及匹配验证）
   - 商品多图轮播、库存实时显示与警告
   - 友好的操作反馈与错误提示
2. **系统安全增强**
   - 密码SHA-256加密存储与验证
   - 登录状态验证（购物车、订单操作需登录）
   - 用户名/手机号唯一性实时检查
   - 严格的权限管理（普通用户与管理员权限分离）
3. **核心功能完善**
   - 完整购物流程：浏览→加购→下单→支付→物流跟踪
   - 库存管理：添加购物车/下单时库存检查，订单成功后自动扣减
   - 订单智能排序：后台按状态与时间优先级展示订单
   - 双搜索框设计：支持关键词快速搜索与高级搜索

#### 安装教程
1. 环境准备
   - JDK 17+、MySQL 8.0+、Tomcat 10.1+
   - 配置MySQL数据库，执行SQL脚本创建用户表、商品表、订单表等核心表
   - 导入项目依赖（Servlet API、MySQL驱动等）
2. 项目部署
   - 克隆仓库：`git clone [仓库地址]`
   - 配置数据库连接信息（修改数据库URL、用户名、密码）
   - 部署项目到Tomcat服务器
   - 启动Tomcat，访问`http://localhost:8080/flower_shop`即可进入系统
3. 初始账号
   - 管理员账号：admin/admin123
   - 测试用户账号：flowerlover/Pwd123456

#### 使用说明
1. 前台操作
   - 注册/登录：填写合法信息完成注册，使用账号密码登录系统
   - 商品浏览：通过分类导航或搜索框查找商品，查看商品详情
   - 购物车管理：添加商品、调整数量、删除商品，确认后结算
   - 订单处理：提交订单后选择支付方式，查看订单状态与物流信息
2. 后台操作
   - 登录后台：通过首页"后台管理"入口，使用管理员账号登录
   - 商品管理：添加/修改/删除商品信息，设置推荐商品
   - 订单管理：查询订单、修改订单状态、处理退款/取消订单
   - 类目管理：添加/编辑/删除商品类目，构建分类体系
